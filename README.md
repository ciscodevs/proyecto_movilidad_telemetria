# App Telemetria

Diseño y desarrollo de una arquitectura de telemetría bidireccional integrada con Firebase y un Gateway en Python. Implementé una aplicación móvil para el control persistente de actuadores físicos (seguros y cajuela) y un protocolo de seguridad activa con sensores de impacto para apertura de emergencia. Optimicé el rendimiento del sistema mediante algoritmos de Soft-Start, logrando estabilidad eléctrica y sincronización de datos en tiempo real entre hardware y nube.
